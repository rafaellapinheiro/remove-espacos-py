# remove-espacos-py

Desenvolvi um pequeno projeto em Python com foco em manipulação de dados textuais, aplicando lógica de programação e estruturas de dados para remover espaços em branco do início e do final de entradas fornecidas pelo usuário.
O desafio consistia em trabalhar com strings; no entanto, optei por uma abordagem alternativa, convertendo a entrada em lista para realizar a manipulação dos caracteres de forma mais controlada, sem utilizar diretamente métodos prontos da linguagem.
